## Страница регистрации и входа
![reg_and_login](https://github.com/user-attachments/assets/5710357d-e5f6-4d79-a99b-f21153cf2650)

При входе пользователь указывает почту и пароль. Если пользователь не зарегистрирован, то по кнопке "Зарегистрироваться" переходит на страницу регистрации.

Пользователь при регистрации указывает свою почту и пароль, а также ФИО и свою роль (арендодатель/арендатор).

## Страница с добавлением объекта 
![create_page](https://github.com/user-attachments/assets/56ed165a-4c8c-4367-9c82-039590d07d31)

Пользователь создаёт объект и вводит основную информацию о нём: адрес, фотографии, описание, контакты, цена.

## Страница с дашбордом объектов недвижимости 
![dashboard](https://github.com/user-attachments/assets/dd96a23e-1171-4252-985f-28142a613797)

На странице показываются объекты недвижимости с основной информации: адрес, фото, цена, лейбл (сдана/не сдана). Для поиска также есть фильтры, они открываются нажатием на иконку.

По нажатию на карточку объекта, открывается его страница.

## Страница личного кабинета
![pay_page](https://github.com/user-attachments/assets/e5c92f42-a109-4e3a-8ef4-f65fe81702ed)

Здесь будет информация о пользователе и его подписке (для арендодателя) или платеже (для арендатора). По нажатию на кнопку "Оплатить" перекидывает на внешний сервис оплаты.
